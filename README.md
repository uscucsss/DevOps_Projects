# My first Docker Project

Это будет моим первым проектом в рамках обучения DevOps. В ходе работы я научился упаковывать статический сайт в docker-контейнер

# Внутри у нас:
* **Nginx** - сам веб-сервер.
* **Docker** - нужен для контейнеризации приложения.
* **Custom HTML** - простая веб-страница.

## Как запустить проект:

1. **Сборка образа:**
docker build -t my-super-site . - точка в конце обязательна 

2. **Запуск контейнера**
docker run -d -p 9090:80--name my-running-site my-super-site

После запуск будет доступен сайт по адресу : 192.168.48:9090
